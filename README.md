# polytryhard

A statistical research lab for prediction markets — strategy research platform for Kalshi weather markets.

**Current status:** Frontend prototype only. All data is mocked in-browser; every control is live-wired with ledger invariants enforced client-side. No FastAPI or Postgres yet.

## Quick start

```bash
cd ui
npm install
npm run dev
```

Open http://localhost:5173
